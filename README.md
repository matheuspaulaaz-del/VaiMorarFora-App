# VaiMorarFora App (MVP)

Projeto criado com Expo + React Native.

## Rodar localmente

1. Instale as dependências:

```bash
npm install
```

2. Configure variáveis de ambiente (Supabase / Stripe) em `app.config` ou `.env` (ainda não incluído).

3. Rode o app:

```bash
npm start
```

Este projeto é um esqueleto inicial com:

- Onboarding (país, inglês, perfil)
- Tela de sugestões
- Paywall estático (simulação)
- Tabs para Guia, Consulado e Empregos
